Class Flow

Provides extension methods to attach approval workflow functionality to entities. This is the main entry point for using the PhilApprovalFlow framework.

GetApprovalFlow<T>(IApprovalFlow<T>)

Attaches the PhilApprovalFlow engine to an entity.

Declaration
public static ICanSetUser GetApprovalFlow<T>(this IApprovalFlow<T> f) where T : IPAFTransition, new()
Parameters
Type Name Description
IApprovalFlow<T> f

The entity to attach the approval flow to.
Returns
Type Description
ICanSetUser

An instance of ICanSetUser that can be used to set the user context and perform actions on the workflow.
Type Parameters
Name Description
T

The type of transition used in the approval flow, which must implement IPAFTransition and have a parameterless constructor.
Examples
// Create an entity that implements IApprovalFlow<YourTransitionType>
var entity = new Invoice();

// Attach the approval flow engine
var workflow = entity.GetApprovalFlow();

// Set the user context and request approval
workflow.SetUserName("john.doe@example.com")
        .RequestApproval("jane.smith@example.com", "Manager", "Please review this invoice");
